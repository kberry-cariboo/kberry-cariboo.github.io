Privacy Policy

1. Overview

CashFlow is a personal cash-flow forecasting and budgeting application developed by Kenneth Berry. The app is hosted on GitHub Pages at https://kberry-cariboo.github.io/. Your privacy is important, and this policy explains what data the app accesses, how it is stored, and what choices you have.

2. Data Storage

CashFlow requires a free account (email and password) to use. Your budget data — entries, categories, overrides, and preferences — is stored in a Supabase project (a hosted Postgres database) that the app’s operator controls, scoped to your account with row-level security so only you and the household members you invite can read or write it. A copy is also cached in your browser’s localStorage for offline use and faster loading.

  • Your password is never stored or seen by CashFlow — authentication is handled entirely by Supabase Auth.
  • Clearing your browser data removes the local cache only; your data remains in Supabase and reloads the next time you sign in.

3. Household Sharing & GitHub Gist (Optional Manual Backup)

You can invite family members to share your budget (a “household”). Each person signs in with their own email and password; invited members can read and write the same shared budget data.

CashFlow also offers an optional manual backup/import feature using a GitHub Gist you create and control, via a GitHub Personal Access Token you generate and paste into the app. This token is stored only in your browser’s localStorage and is sent only to GitHub’s API, directly from your browser. This is a manual, opt-in feature — it is not used for automatic sync.

4. Information We Collect

CashFlow does not collect, store, or transmit any personal information beyond what’s described above. Specifically:

  • No analytics or tracking scripts are used.
  • No cookies are set by the application.
  • No personal data is sent to any server other than Supabase (required) and, if you opt in, GitHub’s Gist API.
  • No advertising networks or other third-party services are integrated.

5. Third-Party Services

CashFlow is a self-contained app — its code, fonts, and icons are hosted alongside it rather than loaded from a CDN. It communicates with the following third-party services:

  • Supabase — authentication and database storage for your budget data (required to use the app).
  • GitHub Gist API — only if you opt into the manual backup/import feature, using a token you provide.
  • Anthropic API — only if you enter your own API key for the optional AI Insights feature; requests go directly from your browser to Anthropic using your key.

These services are governed by their own privacy policies. CashFlow does not control or have access to any data these services may collect.

6. Data Retention & Deletion

  • Local cache: Persists in your browser until you clear it or use the in-app “Reset Local Cache” option; your Supabase data is unaffected either way.
  • Supabase data: Remains in the project until you delete your account or ask the app’s operator to delete it.
  • GitHub Gist data: If you use the optional Gist backup, the Gist remains in your own GitHub account until you delete it there.

7. Children’s Privacy

CashFlow is not directed at children under the age of 13, and we do not knowingly collect personal information from children.

8. Changes to This Policy

This privacy policy may be updated from time to time. Changes will be reflected on this page with an updated effective date.
